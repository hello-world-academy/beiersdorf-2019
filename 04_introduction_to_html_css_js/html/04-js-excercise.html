Coding for Non-Coders

Date: 2019-09-25

In this workshop you learn (some) basics of coding. You get an overview where to use programming skills to improve your day to day workflow and code little applications by youself.

We focus on the following skills:

  • Basic understanding of coding
  • Basic understanding of technologies
  • Teambuilding and Softskills: Collaboration, Critical thinking, Creativity

We, the Hello World Academy, help you to create digital products, processes and buisness models and help you to understand the technicalities of the digitalization.
