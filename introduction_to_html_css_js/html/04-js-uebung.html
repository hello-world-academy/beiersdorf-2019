Grundlagen von Coding für Non-Coder

Datum: 29.05.2019

Im Rahmen des Workshops lernen Sie die Grundlagen rund ums Programmieren kennen. Sie erhalten einen Überblick über unterschiedliche Einsatzmöglichkeiten von Programmiersprachen und machen anhand von Anwendungsbeispielen erste eigene Erfahrungen.

Folgende Fähigkeiten wollen wir vermitteln:

  • Grundlagen von Coding
  • Grundlegendes Verständnis von digitalen Technologien
  • Teambuilding & Softskills: Kollaboration, Kritisches Denken, Kreativität

Wir als Hello World Academy helfen Ihnen digitale Produkte, Prozesse und Geschäftsmodelle zu gestalten und die technische Seite der Digitalisierung zu verstehen.
